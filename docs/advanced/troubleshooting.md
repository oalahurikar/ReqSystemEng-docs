# Troubleshooting

Common issues and solutions when using ReqDrivenDev.

## Requirements Not Appearing

**Problem:** Requirement was created but doesn't appear in Truth column.

**Solutions:**
1. **Refresh the Page:** Sometimes UI needs refresh to show new items
2. **Check Validation:** Requirement may have been blocked - check for error messages
3. **Verify Status:** Requirement may be in Draft status - check status filter

## Search Not Finding Requirements

**Problem:** Semantic search doesn't find requirements you know exist.

**Solutions:**
1. **Use More Specific Terms:** Generic terms may not match well
2. **Check Spelling:** Ensure requirement text is spelled correctly
3. **Wait for Indexing:** New requirements may take a moment to be indexed

## Constraint Violations Not Detected

**Problem:** Requirement violates constraint but system doesn't block it.

**Solutions:**
1. **Check Scenario:** Constraint may apply to different scenario
2. **Verify Constraint Expression:** Ensure constraint expression is correct
3. **Check Constraint Link:** Ensure constraint is properly linked to scenario

## Performance Issues

**Problem:** System is slow or unresponsive.

**Solutions:**
1. **Check Graph Size:** Very large graphs may be slower
2. **Clear Browser Cache:** Sometimes helps with UI performance
3. **Check Network:** Ensure stable connection to server

## Getting Help

If you encounter issues not listed here:

1. **Review Error Messages:** They usually contain specific details
2. **Check Documentation:** Review relevant guides
3. **Contact Support:** Report issues with details about what you were doing
